## ✨ Features

- Elegant and minimal invitation layout  
- Fully responsive (mobile, tablet, desktop)  
- Clear event information section  
- Simple RSVP form with basic client-side validation  
- Smooth hover effects & modern typography  
- Easy to customize colors, fonts, images & content  
- No framework / vanilla HTML5 + CSS3 only  
- Beginner-friendly, well-commented code
