# 🎉 Event Invitation Page

A simple and elegant **Event Invitation Page** built using **HTML and CSS**.  
This project is perfect for beginners to understand webpage structure, styling, and basic form handling.

---

## 📌 Project Overview

This webpage displays an invitation for an event with essential details such as date, time, venue, and an RSVP form.  
It can be customized for birthdays, weddings, college events, or corporate functions.

---

## 🛠️ Technologies Used

- HTML5  
- CSS3  

---

## ✨ Features

- Clean and attractive invitation layout  
- Event details section (Date, Time, Venue)  
- RSVP form with validation  
- Responsive design  
- Beginner-friendly code structure
